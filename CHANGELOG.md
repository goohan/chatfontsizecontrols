# Changelog

All notable changes to this extension are documented in this file.

## 0.0.4

- Expand the default chat font size range from `12–16` to `10–20`.
- Change the default adjustment step from `0.5` to `1`.

## 0.0.3

- Replace the Marketplace icon with a smooth, high-resolution zoom illustration.
- Add attribution for the original icon design.

## 0.0.2

- Add a Marketplace icon consistent with the visual style of the publisher's other extensions.

## 0.0.1

- Add commands to increase, decrease, and reset AI chat font sizes.
- Keep Markdown and code block font sizes synchronized with a configurable offset.
- Add configurable step, limits, reset value, and keyboard shortcuts.
